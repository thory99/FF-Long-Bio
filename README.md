# ⚡ FF Bio Tool (THORYXFF)

A simple and powerful web tool to create and set long bio for your Free Fire profile.

## 🔥 Features

* ✨ Long Bio Support
* 🎨 Stylish Fonts & Symbols
* ⚡ Fast & Easy to Use
* 📱 Mobile Friendly UI
* 🌍 Region Support

## 🚀 How to Use

1. Enter your desired bio text
2. Customize with symbols/fonts
3. Copy the generated bio
4. Paste it into your Free Fire profile

## 🌐 Live Concept

Make your profile unique and stand out with stylish long bio.

## 📌 Configuration

You can customize the site using `config.py`:

* Change site name, logo, and branding
* Update Free Fire version
* Add your social links
* Modify bio character limits
